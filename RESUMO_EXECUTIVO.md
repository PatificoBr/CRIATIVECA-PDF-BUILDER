```
Python 3.13+
```

- **Todas as páginas**: Rodapé com numeração
- **Otimização**: Automática com PyMuPDF

## 📞 SUPORTE RÁPIDO

### Problema: "ModuleNotFoundError"
```bash
pip install -r requirements.txt
```

### Problema: "Nenhuma imagem encontrada"
- Certifique-se que os arquivos são PNG, JPG, JPEG ou WEBP
- Tente novamente com a pasta correta

### Problema: "Permissão negada"
- Verifique permissões da pasta
- Tente como administrador
- Use outra pasta

### Problema: Precisa de ajuda?
- Leia `README.md`
- Veja `GUIA_FINAL.md`
